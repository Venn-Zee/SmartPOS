Smart POS System using Django Backend Framework.
-
Django Framework not available, please download django first via cmd promt: 

py -m pip install Django==6.0.5 or python -m pip install Django==6.0.5 (Linux/macOS)

inside the main folder, type:

python -m venv venv

if a venv folder is added, then its ready to be run

run with
-
venv\Scripts\activate

python manage.py runserver

if venv folder is one tier above the project file, then cd first
